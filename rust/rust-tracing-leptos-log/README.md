# Rust Tracing Using Leptos Log

Make tracing print log with `leptos::logging:log!` macro

## Status

Working

## Getting Started

```bash
cargo install trunk
cargo install leptosfmt # For formatting the leptos view! macro
rustup target add wasm32-unknown-unknown
```

To run

```bash
trunk serve
```
